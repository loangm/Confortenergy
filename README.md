# Confort Energy

Site vitrine pour **Confort Energy**, entreprise de services de rénovation et d’isolation dans le bâtiment. Développé avec Vue 2 & Vuetify.

🔗 [confortenergy.vercel.app](https://confortenergy.vercel.app)

-----

## Stack technique

|Technologie|Version|
|-----------|-------|
|Vue.js     |2.6.11 |
|Vuetify    |2.4.0  |
|Vue CLI    |4.5.0  |
|Sass       |1.32.0 |
|Core-JS    |3.6.5  |
|Node.js    |18.x   |

-----

## Installation & développement local

### Installer les dépendances

```bash
npm install
```

### Lancer en mode développement (hot-reload)

```bash
npm run serve
```

### Compiler pour la production

```bash
npm run build
```

### Linter

```bash
npm run lint
```

> Voir la [Configuration Reference](https://cli.vuejs.org/config/) pour plus d’options.
